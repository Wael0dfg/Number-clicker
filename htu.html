How To Use Number Clicker

1. Basic Counting

Click the Click Me button to increment the counter by 1 each time.

2. Decreasing Count

Use the -1 button to decrease the count by 1 (won't go below 0).

3. Resetting

Press the Clear button to reset the counter back to 0.

4. Saving Progress

Click Save Progress to download your current count as a JSON file.

You can name the file whatever you want when prompted.

5. Loading Progress

Use Load Progress to upload a previously saved JSON file.

This will restore your counter to the saved number.

Tips
  • The counter will never go below 0
  • Saved files are in standard JSON format
  • Works on mobile devices too
  • Your count is only saved when you explicitly save it
